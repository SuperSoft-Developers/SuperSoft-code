# Supersoft Code (.ssc)

A lightweight programming language for games and apps.

Supports:
- 2D graphics (sprite system)
- Window creation
- Simple scripting
- Future: 3DS / Wii export

## Example

```ssc
window 800 600 "My Game"

sprite "player.png" 100 200

## Build
```bash
./sscc main.ssc output.cpp
g++ output.cpp -o game
./sscc main.ssc output.cpp
g++ output.cpp -o game
