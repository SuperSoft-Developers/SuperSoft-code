# 🚀 Supersoft Code (.ssc)

Supersoft Code is a lightweight programming language designed for **games and simple applications**, with a focus on **simplicity, portability, and retro platforms**.

It compiles `.ssc` files into C++ code, which can then be built for Windows, macOS, and later ported to systems like **Nintendo 3DS and Wii**.

---

## 🎮 Features (v0.1)

- 🖥️ Simple scripting syntax
- 🪟 Window creation
- 🎨 Sprite rendering system
- ⚡ Transpiles to C++
- 🔧 Easy CLI compiler (`sscc`)
- 📦 Designed for future console support (3DS / Wii)

---

## 🧠 Philosophy

Supersoft Code aims to be:

- Simple like scripting languages
- Powerful like game engines
- Portable like low-level C++
- Accessible for beginners

Think of it as:
> “Love2D simplicity + C++ performance + retro console dreams”
